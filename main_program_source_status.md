1. 变量名、类名有部分简化/混淆（如 `a/b/c/d`）
2. 注释、原始源码格式、泛型细节有所丢失
3. 某些运行时依赖行为仍依赖 `kdrawLoader.dll`
4. 如果要“可维护源码工程”，还需要人工整理命名和模块边界
